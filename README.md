# Igetis Telematics Billing System

Google Apps Script система учёта телематики: синхронизация с API, выставление счетов, дашборд, аудит.

## Файлы

| Файл | Описание |
|------|----------|
| `Code.gs` | Основной скрипт — вся логика |
| `InvoiceSidebar.html` | Форма создания счетов |
| `ImportSidebar.html` | Импорт авто и плательщиков |
| `SearchSidebar.html` | Поиск по VIN и трекеру |
| `igetis_import_template.xlsx` | Шаблон для импорта данных |

## Установка

1. Google Sheets → Расширения → Apps Script
2. Вставить содержимое `Code.gs` в редактор
3. Создать HTML-файлы: `InvoiceSidebar`, `ImportSidebar`, `SearchSidebar`
4. Вставить токен Igetis в переменную `IGETIS_TOKEN`
5. Меню → Телематика → Первичная настройка листов

## Структура листов

- `🏢 Плательщики` — контрагенты (API + ручные, ID: MAN-YYYYMMDD-XXXX)
- `🚛 Автомобили` — ТС (синхронизация с API, ручное добавление)
- `🧾 Счета` — выставленные счета с VAT 22%
- `📊 Дашборд` — сводка по статусам
- `📋 Аудит` — отчёт о проблемах (генерируется)
